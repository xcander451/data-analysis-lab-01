# Лабораторная работа №1: Первичное исследование и оценка качества данных

## Описание
Этот проект посвящен первичному анализу (EDA) набора данных банковского маркетинга. Цель — изучить структуру данных, выявить проблемы качества (пропуски, дубликаты, выбросы) и наметить план по их очистке.

## Данные
Используется датасет Bank Marketing Data Set.
Файл: data/bank-additional-full.csv
Данные содержат информацию о клиентах банка и результатах маркетинговой кампании (оформление срочного депозита).

## Структура репозитория
- notebooks.ipynb Jupyter ноутбук с кодом анализа.
- report/quality_report.md Отчет о качестве данных и план действий.
- data/ Исходные данные.
